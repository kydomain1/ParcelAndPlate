# ParcelAndPlate - Lifestyle Blog Website

A modern, minimalist lifestyle blog website featuring six main categories: Clothing & Accessories, Health & Beauty, Home & Garden, Travel & Accommodation, Finance & Insurance, and Food & Beverage.

## Features

- **Modern Design**: Aurora gradient effects, transparent backgrounds, and smooth animations
- **Responsive Layout**: Works seamlessly on desktop, tablet, and mobile devices
- **Category Filtering**: Filter articles by category
- **Search Functionality**: Search articles by title, excerpt, or category
- **Pagination**: Easy navigation through article pages
- **Article Pages**: Detailed article view with recommended products
- **Product Pages**: Clean product showcase with details

## File Structure

```
ParcelAndPlate/
├── index.html          # Home page
├── article.html        # Article detail page
├── product.html        # Product detail page
├── css/
│   └── style.css       # Main stylesheet
├── js/
│   ├── data.js         # Sample data (articles & products)
│   ├── main.js         # Main functionality
│   ├── article.js      # Article page functionality
│   └── product.js      # Product page functionality
└── README.md           # This file
```

## Usage

1. Open `index.html` in a web browser to view the homepage
2. Navigate through articles using category filters or search
3. Click on any article to view the full article page
4. Click on recommended products to view product details

## Design Features

- **Aurora Gradient**: Beautiful gradient effects throughout the site
- **Transparent Backgrounds**: Modern glassmorphism effect
- **Smooth Animations**: Hover effects and transitions
- **Modern Typography**: Inter font family for clean, readable text
- **Minimalist Layout**: Clean design with ample white space

## Categories

1. **Clothing & Accessories** - Fashion and style articles
2. **Health & Beauty** - Skincare and wellness content
3. **Home & Garden** - Interior design and gardening tips
4. **Travel & Accommodation** - Travel guides and destination reviews
5. **Finance & Insurance** - Financial planning and investment advice
6. **Food & Beverage** - Recipes and culinary experiences

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Customization

To add more articles or products, edit the `js/data.js` file and add new entries to the `articles` or `products` arrays.

## Notes

- This is a static website (no backend required)
- Images are loaded from Unsplash (placeholder URLs)
- All functionality is client-side JavaScript
